Site Vars

Name Description Value
site.time Current time of jekyll build {{site.time| jsonify}}
site.pages A list of all Pages. array_length: {{site.pages|size}}
site.posts A reverse chronological list of all Posts. array_length: {{site.posts|size}}
site.related_posts If the page being processed is a Post, this contains a list of up to ten related Posts. {{site.related_posts| jsonify}}
site.static_files A list of all static files (i.e. files not processed by Jekyll's converters or the Liquid renderer). array_length: {{site.static_files|size}}
site.html_pages A subset of `site.pages` listing those which end in `.html`. array_length: {{site.html_pages|size}}
site.html_files A subset of `site.static_files` listing those which end in `.html`. {{site.html_files| jsonify}}
site.collections A list of all the collections. array_length: {{site.collections|size}}
site.data A list containing the data loaded from the YAML files located in the _data directory. {{site.data| jsonify}}
site.documents A list of all the documents in every collection. array_length: {{site.documents|size}}
site.url Contains the url of your site as it is configured in the _config.yml. {{site.url| jsonify}}

Page Vars

Name Description Value
page.content The content of the Page. {{page.content| jsonify}}
page.title The title of the Page. {{ page.title | jsonify}}
page.excerpt The un-rendered excerpt of a document. {{page.excerpt| jsonify}}
page.url The URL of the Post without the domain, but with a leading slash {{page.url| jsonify}}
page.date The Date assigned to the Post. {{page.date| jsonify}}
page.id An identifier unique to a document in a Collection or a Post {{page.id| jsonify}}
page.categories The list of categories to which this post belongs. {{page.categories| jsonify}}
page.tags The list of tags to which this post belongs. {{page.tags| jsonify}}
page.path The path to the raw post or page. {{page.path| jsonify}}
page.next The next post relative to the position of the current post in site.posts. {{page.next| jsonify}}
page.previous The previous post relative to the position of the current post in site.posts. {{page.previous| jsonify}}
